# B02010 システムコンテキストダイアグラム

## 1. 本書の位置付け

本書は「書籍管理Webアプリ」（以下、本システム）の**システムコンテキスト**を定義する。本システムと、その外部に存在する利用者・外部要素との関係を一覧化し、システム境界（スコープ）を明示することを目的とする。

本書は [B01010 システム振舞い共通ルール](../010_要件定義/B01010_システム振舞い共通ルール.md) を前提とし、以下成果物の基礎入力となる。

- B02020 アクタ定義/ロール定義
- B02030 ユースケース図
- B02040 ユースケース記述
- A03110 ソフトウェア論理構成（基本設計）

## 2. 前提

| 項目         | 内容                                                                 |
| ------------ | -------------------------------------------------------------------- |
| システム名   | 書籍管理Webアプリ                                                    |
| 利用形態     | 個人Windows PC上で利用するWebアプリ（ローカル起動）                  |

| 実行環境     | Node.js 24 LTS                                                       |
| 外部連携     | なし（インターネット経由の外部システム連携を行わない）               |

## 3. システム境界（スコープ）

本システムの**システム境界**は、ユーザのローカルPC上で起動する単一のWebアプリ（Express v5 + EJS v3 + better-sqlite3 v11）とそのデータファイル（SQLite）の組合せである。

- **境界内**: Express サーバプロセス、EJS テンプレート、SQLite データファイル（`data/books.db`）、Web ブラウザに配信される HTML/CSS/JS。
- **境界外**: ユーザ本人、Web ブラウザ（クライアントとしてのみ）、PCのファイルシステム。
- **本システムが連携する外部システム**: **存在しない**。インターネット越しの API・認証基盤・メール送信・クラウドストレージ等の利用は対象外。

## 5. 入出力（インタフェース）一覧

本システムが境界をまたいでやり取りする入出力を網羅する。

| ID    | 方向     | 起点          | 終点            | 内容                                       | プロトコル/形式            |
| ----- | -------- | ------------- | --------------- | ------------------------------------------ | -------------------------- |
| IF-01 | IN       | ユーザ        | Webブラウザ     | キーボード/マウスによる画面操作            | OS のヒューマンインタフェース |
| IF-02 | IN/OUT   | Webブラウザ   | Expressサーバ   | 画面要求・フォーム送信・JSON取得（任意）   | HTTP/1.1（localhost）      |
| IF-03 | OUT      | Expressサーバ | Webブラウザ    | HTML（EJSレンダリング結果）/ 静的アセット  | HTTP/1.1（localhost）      |
| IF-04 | IN/OUT   | Expressサーバ | SQLiteファイル  | 書籍データの CRUD                          | better-sqlite3（同期API）  |
| IF-05 | OUT      | Webブラウザ   | ユーザ          | 画面表示・通知バー・ダイアログ             | 画面描画                   |

**境界外への通信は IF-01〜IF-05 のみ**で、これ以外（メール送信・外部 API・クラウド連携等）は存在しない。

## 6. 利用シーンと前提条件

| 項目             | 内容                                                                 |
| ---------------- | -------------------------------------------------------------------- |
| 起動方法         | ユーザがコマンドラインから `node` 等で本システムを起動する。         |
| アクセス先       | ブラウザで `http://localhost:3000/` を開く（ループバックのみ）。     |
| ネットワーク要件 | ローカルループバックで完結。外部ネットワーク接続は不要。             |
| マルチユーザ     | 想定しない。同時アクセスは1ブラウザを基本とする。                    |
| バックアップ     | `data/books.db` を手動コピーで取得（運用は利用者の自己責任）。       |

---

## 7. スコープ外事項

以下は本システムのスコープ外であり、システムコンテキストには含めない。

- ログイン認証・ユーザ管理・パスワードリセット
- 複数ユーザ／ロール権限制御
- 外部 API 連携（書誌情報の自動取得等）
- メール送信・通知サービス
- クラウドストレージ・他端末同期
- モバイル端末向け最適化
- 多言語対応
